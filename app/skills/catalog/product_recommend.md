name: product_recommend
description: 按品牌、价格、品类和使用场景筛选商品，并输出推荐结果。
---

# 目标
当用户想要推荐商品时，提取品牌、价格上限、品类、使用场景等条件，然后调用 `data_center_search_products`。

# 适用问题
- 推荐 500 元以内李宁鞋
- 推荐适合通勤的跑鞋
- 推荐指定品牌和价格区间的商品

# 执行要求
1. 尽量从用户问题中提取这些字段：
   - `brand`
   - `category`
   - `price_max`
   - `use_case`
2. 默认参数：
   - `limit` 默认 `5`
   - `sort_by` 默认 `score`
3. 如果用户已经给出足够条件，直接调用工具
4. 如果用户只说“推荐一下商品”而没有任何限定条件，可以先追问品牌、预算或用途
5. 如果用户说“多少钱”而不是“推荐”，不要使用这个 skill

# 输出要求
- 明确说明筛选条件
- 推荐 3 到 5 个候选
- 每个候选尽量包含价格、库存和推荐理由
- 说明数据来自 mock 商品服务

# 输出要求
- 返回推荐商品列表
- 明确筛选条件
- 每个候选要有推荐理由
- 说明当前是 mock 数据
