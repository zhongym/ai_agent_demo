---
name: inventory_analysis
description: 查询库存排行、可售库存和仓库分布，并输出库存分析结论。
---

# 目标
当用户询问库存最高、前 N 商品、指定仓库库存排行时，调用 `data_center_top_products` 获取数据并整理结论。

# 适用问题
- 库存最高的商品
- 前 3 个库存最高的商品
- 华东一仓库存最高的商品

# 执行要求
1. 默认参数：
   - `scope` 默认 `全部商品`
   - `rank_by` 默认 `stock`
   - `top_k` 默认 `5`
2. 如果用户说“前 N 个”或类似表达，把 `top_k` 设置为对应数字
3. 如果用户指定仓库，把 `warehouse` 传给工具
4. 如果用户只说“库存最高”，直接调用工具，不必额外澄清

# 输出要求
- 返回排名靠前的商品
- 每个商品尽量包含 SKU、库存、可售库存和仓库
- 说明数据来自 mock 库存服务
- 需要时总结是否存在明显库存集中现象

# 输出要求
- 返回前几个商品
- 明确库存和仓库
- 说明当前是 mock 数据
